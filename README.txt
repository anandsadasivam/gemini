Access the REST endpoints: Once your Spring Boot application is running, you can access the REST services using a browser, Postman, curl, or any other HTTP client:

http://localhost:8080/hello (or http://localhost:8080/hello?name=YourName)
http://localhost:8080/greet
http://localhost:8080/add?a=5&b=3

--for run Spring Boot Application

$ mvn spring-boot:run
